HTML/CSS

El guardado en github de archivos debe ser en minúsculas así:index.html y styles.css

Si son archivos apartes en el html deberás referenciar al css según lo hayas guardado:

<link rel="stylesheet" href="styles.css">

<link rel="stylesheet" href=styles.css >

También existe una código referencial responsive y es este:

<link rel="stylesheet" media="all" href= styles.css />

SQL

A continuación las consultas select mas usadas: SQL

  1. use nombre_base
  2. select * from.dbo.tblClientes
  3. delete from tabla/ borrar datos de tabla
  4. drop table nombre_tabla/ eliminar tabla
  5. drop database nombre_base/ eliminar una base de datos
  6. select columna1, columna2 from tabla
  7. select * from table WHERE ColumnaID=1;
  8. select * from table WHERE Country='Mexico';
  9. select first_name,last_name from Customers where not last_name='Reinhardt';
  10. delete from table_name; borra solo los registros  mantiene las columnas y la tabla
  11. Truncate from table;borra todos los datos de la tabla sin condición
  12. alter table table_name; borrado permanente
  13. drop column column_name;borra la columna de una tabla permanentemente
