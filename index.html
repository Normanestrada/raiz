POWER BI

importando tablas de excel desde BI

  1. Abre Power BI Desktop y selecciona "Obtener datos" desde la pestaña "Inicio" en la cinta de opciones.
  2. En el cuadro de diálogo "Obtener datos", selecciona "Excel" y luego elige el archivo de Excel que contiene la tabla que quieres importar.
  3. Selecciona la hoja de Excel que contiene la tabla que deseas importar y haz clic en "Cargar" para comenzar el proceso de importación.
  4. Power BI Desktop detectará automáticamente la tabla de Excel y te mostrará una vista previa de los datos. Si todo parece correcto, haz clic en "Cargar" para importar la tabla a Power BI.
  5. Después de cargar la tabla, puedes comenzar a trabajar con ella en Power BI Desktop. Puedes crear visualizaciones, agregar filtros y crear informes y paneles para compartir con otros usuarios.

Creando relaciones de tablas en power BI

  1. Abre el modelo de datos.
  2. Selecciona ambas tablas.
  3. Haz clic en la opción "Administrar relaciones".
  4. Crea una nueva relación.
  5. Establece las columnas relacionales.
  6. Configura la cardinalidad.
  7. Guarda la relación.
  8. Verifica la relación.
  9. Utiliza la relación.
  10. En resumen, la lógica detrás de las relaciones en Power BI es conectar dos tablas a través de una columna común para combinar información de ambas tablas en una sola visualización o análisis. La forma en que se establece la relación dependerá de la estructura de los datos y de los requisitos específicos de análisis Ejemplo tabla principal cliente ID venta, nombre, cédula ha tabla transaccional ventas, cantidad, en análisis importante irá a la tabla transaccional
  11. Se da automática al cargar y cumple requisitos como mismos encabezados y contenido similar tipo de datos número, texto,etc. De lo contrario hay que relacionar manualmente.

Linked server


- Vincular servidores de la Data Warehouse que pueden ser como :  MySQL, Oracle, soluciones en la nube con: Amazon Redshift, Microsoft Azure, Google cloud 

- Dif entre  Oracle y sql: Integración: SQL Server está diseñado para integrarse con otras herramientas de Microsoft, como Excel, Power BI y Visual Studio. 
  Oracle también ofrece integración con algunas de estas herramientas, pero no es tan estrecha como la de SQL Server con las herramientas de Microsoft.

 -Plataforma: Oracle se ejecuta en una amplia variedad de plataformas, incluyendo Unix, Linux y Windows. SQL Server solo se ejecuta en plataformas de Microsoft Windows.
  crece más a nivel horizontal  por eso resulta muy conveniente unir ambas plataformas

Lenguaje naturak

Para habilitar la función Q&A, sigue estos pasos:

- Abre tu informe de Power BI y selecciona una visualización.
- En la barra de herramientas superior, selecciona "Preguntas y respuestas" en el menú "Visualizaciones".
- Escribe tu pregunta en el cuadro de texto y presiona Enter.
- Power BI intentará entender tu pregunta y presentará una visualización que responda a la pregunta.
- Ten en cuenta que para que la función Q&A funcione correctamente, los datos deben estar bien estructurados y etiquetados, 
  y los nombres de las columnas deben ser descriptivos y coherentes. Si tienes problemas para obtener respuestas precisas a través de la función Q&A, 
  es posible que debas revisar la calidad y consistencia de los datos.
