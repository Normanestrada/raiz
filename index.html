Todo sobre HTML y CSS.

Todo archivo en GitHub desde cero se guarda en minúscula, es decir, el archivo se llamaría index.html y styles.css. No es necesario que los nombres lleven comillas, pero es buena práctica usar comillas en la referencia al archivo CSS. En caso de no usar comillas, la referencia a la hoja de estilo se vería así:

styles.css

En caso de usar comillas, la referencia a la hoja de estilo se vería así:

styles.css
